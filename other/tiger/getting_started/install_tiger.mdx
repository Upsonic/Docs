Upsonic Tiger is a part of [Upsonic Client](/client). In this document, we will guide you through the installation process of the Upsonic Client.

To get started, ensure you have:

- The **Python = > 3.8** installed and operational
- A functional pip3 or pip command linked to your **Python = > 3.8** installation

# Install
You can install the Upsonic Client with a single **pip** command:
```console
pip install upsonic
```
