For creating your own tiger you should install a Upsonic Server docker container. Its will give a dashboard for viewing your own tools and will make documentation automatic. After that you should use the Upsonic Client to connect your Server for this you should get the connection code from your dashboard and finaly you use the tiger function in upsonic client.

[Installation document](https://docs.upsonic.co/server/getting_started/install_on_prem)
