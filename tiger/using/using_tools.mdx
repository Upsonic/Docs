
The Upsonic Tiger is an AI&Computer interface just like human&computer interface. With this Upsonic gives an opportunity to your Agents to make real world operations and tasks for humans. You can use Tiger for get your `Google Calendar` or moving forward your `RC car`. The main princible is on Upsonic core and its own function storage infrastructure.


## Using Tools

You can `load` any tool and use it. Please view the tools setion for viewing all tools. Also you can create your own tiger via On-Prem docker container.

```python
from upsonic import Tiger

google_search_results = Tiger().load("search.google")("Upsonic co")

print(google_search_results)
```
