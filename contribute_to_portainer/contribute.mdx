

## Reporting bugs

If you find a bug, please tell us so we can triage it. All bugs are managed in this [GitHub repo](https://github.com/Upsonic/Server/issues). Before you report a bug, please check of in case someone else has already reported it.



## Feature requests

You can request new features by posting an Idea in our [GitHub Discussions]() forum. Please check to see if someone has already requested the feature you want, and give it an upvote if so.

Learn how we prioritize feature development [in this knowledge base article]().

## Contributing to the Upsonic CE codebase

The Upsonic CE codebase is available in [GitHub](https://github.com/upsonic/server). Please follow our build instructions and [contribution guidelines]() when making a contribution.
