---
title: "In-Memory"
description: "Fast, non-persistent storage for testing and development"
---

## Usage

```python
from upsonic import Agent, Task
from upsonic.durable import DurableExecution, InMemoryDurableStorage

# Create storage
storage = InMemoryDurableStorage()

# Create durable execution
durable = DurableExecution(storage=storage)

# Create agent and task
agent = Agent("openai/gpt-4o")
task = Task(
    description="Quick test task",
    durable_execution=durable
)

# Execute with automatic checkpointing
try:
    result = agent.do(task)
    print(result)
except Exception as e:
    # Resume from checkpoint
    result = agent.continue_durable(
        durable_execution_id=task.durable_execution_id,
        storage=storage
    )
    print(result)
```

## Params

No configuration parameters required.

## Characteristics

- ⚡ Fastest performance
- ❌ No persistence (lost on restart)
- ✅ Perfect for testing
- ✅ No setup required
