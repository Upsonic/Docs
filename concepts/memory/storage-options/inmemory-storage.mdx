---
title: "InMemoryStorage"
description: "Fast in-memory storage for development and testing"
---

## Usage

```python
from upsonic.storage.providers.in_memory import InMemoryStorage

# Basic in-memory storage
storage = InMemoryStorage()

# With LRU cache limits
storage = InMemoryStorage(
    max_sessions=100,
    max_profiles=50
)
```

## Params

- `max_sessions: Optional[int]` - Optional limit on stored sessions (LRU cache)
- `max_profiles: Optional[int]` - Optional limit on stored profiles (LRU cache)
