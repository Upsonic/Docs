## Overview

The `PromptLayer` class connects to [PromptLayer](https://promptlayer.com) via its REST API. Every agent run is automatically logged with the full system prompt, tool definitions, tool calls, token usage, cost, and latency. You can also log evaluation results from accuracy, reliability, and performance evaluators.

<Info>
No extra dependencies required — PromptLayer uses `httpx` which is already included.
</Info>

## Setup

Get your API key from the [PromptLayer dashboard](https://promptlayer.com) → Settings → API Keys.

Set it as an environment variable:

```bash
export PROMPTLAYER_API_KEY=pl_...
```

Or pass it directly in code (see below).

## Usage

### Agent Run

```python
from upsonic import Agent, Task
from upsonic.integrations.promptlayer import PromptLayer
from upsonic.tools.config import tool

@tool(docstring_format="google")
def calculate_discount(price: float, discount_percent: float) -> float:
    """Calculate the final price after applying a discount.

    Args:
        price: Original price
        discount_percent: Discount percentage to apply

    Returns:
        Final discounted price
    """
    return price * (1 - discount_percent / 100)

pl = PromptLayer()

agent = Agent(
    "openai/gpt-4o",
    system_prompt="You are a shopping assistant.",
    tools=[calculate_discount],
    promptlayer=pl,
)

task = Task(description="Calculate 20% discount on $100")
agent.print_do(task)

pl.shutdown()
```

In PromptLayer you will see:
- **System prompt** — the full built prompt (including date, defaults, your custom text)
- **Tools** — `calculate_discount` with its schema
- **Tool calls** — the actual invocation with arguments
- **Time, cost, input/output tokens** — all populated automatically

### Accuracy Evaluation

```python
from upsonic import Agent, Task
from upsonic.integrations.promptlayer import PromptLayer
from upsonic.eval import AccuracyEvaluator

pl = PromptLayer()

agent = Agent("openai/gpt-4o", promptlayer=pl)
judge = Agent("openai/gpt-4o")

evaluator = AccuracyEvaluator(
    judge_agent=judge,
    agent_under_test=agent,
    query="What is the capital of France?",
    expected_output="Paris",
    num_iterations=1,
    promptlayer=pl,
)

import asyncio
result = asyncio.run(evaluator.run())
print(f"Score: {result.average_score}/10")

pl.shutdown()
```

The agent run and the evaluation result are logged separately — the agent logs its own run (with system prompt, tokens, etc.), and the evaluator logs a summary with the score.

### Reliability Evaluation

```python
from upsonic import Agent, Task
from upsonic.integrations.promptlayer import PromptLayer
from upsonic.eval import ReliabilityEvaluator
from upsonic.tools.config import tool

@tool(docstring_format="google")
def get_weather(city: str) -> str:
    """Get the weather for a city.

    Args:
        city: City name

    Returns:
        Weather description
    """
    return f"Sunny in {city}"

pl = PromptLayer()

agent = Agent("openai/gpt-4o", tools=[get_weather], promptlayer=pl)
task = Task(description="What is the weather in Tokyo?")
agent.print_do(task)

evaluator = ReliabilityEvaluator(
    expected_tool_calls=["get_weather"],
    agent_under_test=agent,
    promptlayer=pl,
)
result = evaluator.run(task)
print(f"Passed: {result.passed}")

pl.shutdown()
```

### Performance Evaluation

```python
from upsonic import Agent, Task
from upsonic.integrations.promptlayer import PromptLayer
from upsonic.eval import PerformanceEvaluator

pl = PromptLayer()

agent = Agent("openai/gpt-4o", promptlayer=pl)
task = Task(description="Say hello")

evaluator = PerformanceEvaluator(
    agent_under_test=agent,
    task=task,
    num_iterations=3,
    warmup_runs=1,
    promptlayer=pl,
)

import asyncio
result = asyncio.run(evaluator.run())
print(f"Avg latency: {result.latency_stats['average'] * 1000:.0f}ms")

pl.shutdown()
```

## Prompt Registry

Load versioned prompt templates from PromptLayer and use them as system prompts:

```python
from upsonic import Agent
from upsonic.integrations.promptlayer import PromptLayer

pl = PromptLayer()

agent = Agent(
    "openai/gpt-4o",
    system_prompt=pl.get_prompt("my-agent-v2"),
    promptlayer=pl,
)
agent.print_do("Hello!")
pl.shutdown()
```

## What You See in PromptLayer

| Dashboard Section | What It Shows |
|---|---|
| **Live Logs** | Each agent run or eval as a request with input, output, system prompt |
| **Tools** | Tool definitions sent to the model and actual tool calls made |
| **Time** | Request start/end time and total duration |
| **Cost** | Monetary cost computed from token usage |
| **Input/Output Tokens** | Token counts for prompt and completion |
| **Model Parameters** | Temperature, max_tokens, etc. when explicitly configured |
| **Score** | Evaluation scores (accuracy, reliability, performance) |
| **Metadata** | Custom key-value pairs (eval type, pass/fail, etc.) |
| **Tags** | `upsonic-agent`, `agent:<name>`, `upsonic-eval`, `accuracy-eval`, etc. |

| Parameter | Type | Default | Description |
|---|---|---|---|
| `api_key` | `str` | env `PROMPTLAYER_API_KEY` | PromptLayer API key (`pl_...`) |
| `base_url` | `str` | `https://api.promptlayer.com` | Custom API base URL |
