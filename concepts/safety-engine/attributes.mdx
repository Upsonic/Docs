## Attributes

The Safety Engine is configured through the `Policy` class, which provides the following attributes:

| Attribute | Type | Default | Description |
|-----------|------|---------|-------------|
| `name` | str | (required) | Policy name |
| `description` | str | (required) | Policy description |
| `rule` | RuleBase | (required) | Rule to detect specific content |
| `action` | ActionBase | (required) | Action to take when content is detected |
| `language` | str | `"en"` | Default language for the policy |
| `language_identify_llm` | Any \| None | `None` | LLM for language identification |
| `base_llm` | Any \| None | `None` | LLM for base operations |
| `text_finder_llm` | Any \| None | `None` | LLM for text finding operations |
| `language_identify_model` | str \| None | `None` | Model name for language identification |
| `base_model` | str \| None | `None` | Model name for base operations |
| `text_finder_model` | str \| None | `None` | Model name for text finding |

## Configuration Example

```python
from upsonic import Agent, Task
from upsonic.safety_engine.policies.pii_policies import PIIAnonymizePolicy

# Use pre-built policy
agent = Agent(
    model="openai/gpt-4o",
    user_policy=PIIAnonymizePolicy,  # Anonymize PII in user input
    agent_policy=PIIAnonymizePolicy  # Anonymize PII in agent output
)

# Create task with PII
task = Task(
    description="Process this: email john@example.com, phone 555-1234"
)

# Execute - PII will be automatically anonymized
result = agent.do(task)
print(result)  # PII like email and phone are anonymized
```
