# <div class="hidden">Introduction</div>

The profiles are different assistant actualy. So you can seperate sessions or personnas. 


### remote.profiles

This function change the profile, profiles defined as strings.

```python
remote.profile("default")
```

### remote.reset_memory

When you want to make an cleaning and a clear start, you can use the `reset_memory` function as well.

```python
remote.reset_memory()
```
