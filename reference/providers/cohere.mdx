## Parameters

| Parameter                          | Type                                                  | Default    | Description                                                                                                                                                                                                                      |
| ---------------------------------- | ----------------------------------------------------- | ---------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `api_key`                          | `str \| None`                                         | `None`     | The API key to use for authentication, if not provided, the `CO_API_KEY` environment variable will be used if available.                                                                                                          |
| `cohere_client`                   | `AsyncClientV2 \| None`                               | `None`     | An existing [AsyncClientV2](https://github.com/cohere-ai/cohere-python) client to use. | `http_client`                      | `httpx.AsyncClient \| None`                           | `None`     | An existing `httpx.AsyncClient` to use for making HTTP requests.                                                                                                                                                                 |

## Functions

### `__init__`

Create a new Cohere provider.

**Parameters:**

- `api_key` (str | None): The API key to use for authentication, if not provided, the `CO_API_KEY` environment variable will be used if available.
- `cohere_client` (AsyncClientV2 | None): An existing [AsyncClientV2](https://github.com/cohere-ai/cohere-python) client to use. If provided, `api_key` and `http_client` must be `None`.
- `http_client` (httpx.AsyncClient | None): An existing `httpx.AsyncClient` to use for making HTTP requests.

### `name`

Get the provider name.

**Returns:**

- `str`: The provider name ('cohere')

### `base_url`

Get the base URL for the provider API.

**Returns:**

- `str`: The base URL for the provider API

### `client`

Get the client for the provider.

**Returns:**

- `AsyncClientV2`: The Cohere client

### `model_profile`

Get the model profile for the named model, if available.

**Parameters:**

- `model_name` (str): The name of the model

**Returns:**

- `ModelProfile | None`: The model profile for the named model, if available
