## Functions

### `anthropic_model_profile`

Get the model profile for an Anthropic model.

**Parameters:**

- `model_name` (str): The name of the Anthropic model

**Returns:**

- `ModelProfile | None`: The model profile for the Anthropic model, or None if no specific profile is defined

**Description:**

This function returns the model profile for an Anthropic model. It creates a ModelProfile instance with custom thinking tags ('<thinking>', '</thinking>') that are specific to Anthropic's model format, replacing the default ('<think>', '</think>') tags.
