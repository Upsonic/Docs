## Functions

### `qwen_model_profile`

Get the model profile for a Qwen model.

**Parameters:**

- `model_name` (str): The name of the Qwen model

**Returns:**

- `ModelProfile | None`: The model profile for the Qwen model, or None if no specific profile is defined

**Description:**

This function returns the model profile for a Qwen model. For models starting with 'qwen-3-coder', it returns an OpenAIModelProfile with specific configurations including InlineDefsJsonSchemaTransformer, disabled tool choice required, disabled strict tool definition, and enabled streamed leading whitespace ignoring. For other Qwen models, it returns a basic ModelProfile with InlineDefsJsonSchemaTransformer and enabled streamed leading whitespace ignoring.
