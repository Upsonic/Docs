## Classes

### UserTraits

A comprehensive schema to capture a user's profile, preferences, and in-session state. This model is used by the AI to tailor its responses and maintain long-term context about the user.

#### Parameters

| Parameter                          | Type                                                  | Default    | Description                                                                                                                                                                                                                      |
| ---------------------------------- | ----------------------------------------------------- | ---------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `detected_expertise`               | `Optional[str]`                                       | `None`     | The user's expertise level on the main topic of conversation, e.g., 'beginner', 'intermediate', 'expert'                                                                                                                         |
| `detected_tone`                    | `Optional[str]`                                        | `None`     | The user's preferred communication tone, inferred from their language, e.g., 'formal', 'casual', 'technical'                                                                                                                    |
| `inferred_interests`               | `Optional[List[str]]`                                 | `None`     | A list of general topics or keywords the user seems interested in based on the dialogue                                                                                                                                          |
| `current_goal`                     | `Optional[str]`                                       | `None`     | A concise summary of the user's immediate objective in this specific conversation. What are they trying to accomplish right now? E.g., 'debug a function', 'draft a marketing email', 'understand quantum mechanics'        |
| `session_sentiment`                | `Optional[str]`                                        | `None`     | The user's dominant emotional state in the last few messages. E.g., 'curious', 'frustrated', 'pleased', 'confused', 'neutral'                                                                                                     |
| `communication_style`              | `Optional[List[str]]`                                 | `None`     | Inferred preferences for how the user wants information presented. E.g., 'prefers concise answers', 'requests code examples', 'likes bullet points', 'enjoys detailed explanations', 'asks follow-up questions'                |
| `key_entities`                     | `Optional[Dict[str, str]]`                            | `None`     | A dictionary of important, specific named entities (people, projects, tools) mentioned by the user, along with their context. E.g., {'ProjectTitan': 'A key project the user is working on', 'JaneDoe': 'A colleague the user mentioned'} |
| `long_term_objective_summary`     | `Optional[str]`                                       | `None`     | A running summary of the user's overarching goals or interests observed across multiple sessions. This should be updated, not replaced, to build a long-term understanding                                                      |
