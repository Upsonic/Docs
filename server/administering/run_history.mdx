Upsonic Server keeps the metrics of your function calls. With that you will be able to forecast your usages and track your current status.

## Details of an Run

Upsonic keeps a lot of information about your call, you can access in the `Analyze` button of an call.

- The user
- CPU Usage for One Core
- Ram Usage
- Total Time
- Date (in UTC)
- Machine OS
- Python Version
- Arguments
- Exceptions
